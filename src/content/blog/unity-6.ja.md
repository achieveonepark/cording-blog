Unity 6 はかなり多くの変化を入れようとしている一方で、バージョン表記についてもかなりの混乱を生んでいる……

6000.0 が LTS として出た時は、最初期バージョンなので安定性に不安があり、6000.0 を使うくらいなら 6000.2 を勧めるという意見がかなりあった。

6000.3.0 がリリースされた時も、まだ十分安定していないから引き続き 6000.2 の方がいい、という意見が多かった……

## 今は……？ 6000.3

Toolbar カスタマイズ API が完全に変更されて、以前 Editor UI にボタンを作って使っていたコードは動かなくなってしまった。

決定的なのは、これまで Unity がサポートしていなかった部分をサポートし始めたことで、以前は触りにくかった API 群が整理されてきたことだと思う……

### Unity AI

ついに Unity AI が 6000.3 バージョンから対応すると聞いた。
