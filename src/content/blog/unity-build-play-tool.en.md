# What Is This?

Transferring a Unity build to the right device for each environment is surprisingly tedious.
This is a tool I built to install and run builds with a single click.
I made it by mixing Claude and Codex — it came out better than I expected... wow...

# Quick Look

![image](/images/1774359786310.png)
![image](/images/1774360163553.png)

* Android: Connect a device, press Refresh to detect it, then select a build to install
* iOS: Opens the Xcode project
* WebGL: Starts a local server at localhost:port for testing
